# URL Structure

Correct:

```text
/tents

/tents/the-villa-tent

/tents/noor-mahal

/projects

/projects/oberoi-resort

/blogs

/blogs/luxury-glamping-trends
```

Avoid:

```text
/page?id=12

/project-details?id=45
```

---

# Meta Title Rules

Length:

50–60 Characters

Example:

```text
Luxury Tent Manufacturer | The Villa Tent
```

---

# Meta Description Rules

Length:

140–160 Characters

Example:

```text
Explore premium luxury tents, safari tents, and hospitality solutions designed for resorts, hotels, and tourism projects worldwide.
```

---

# Heading Structure

Only one H1 per page.

Example:


# Image SEO

Every image must have:

* Alt Text
* Optimized File Name
* WebP Format

Correct:

```text
luxury-resort-tent.webp
```

Alt:

```text
Luxury Resort Tent by The Villa Tent
```

Avoid:

```text
IMG_1234.jpg
```
